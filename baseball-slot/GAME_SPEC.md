# 全壘打投注機（Home Run Slot）

純機率棒球投注機台。玩家從主審視角看比賽，下注後投手投球、打者揮棒，球飛向看台不同分區決定賠率。

---

## 核心玩法

1. 玩家選投注額（10 / 50 / 100 / 500）
2. 按「投球」→ 投手投球 → 打者自動揮棒（純機率，不操作擊球）
3. 球飛向 12 個落點之一
4. 依落點結算籌碼 → 回到下注狀態

**重點**：打歪 / 沒打中 → 0 賠率；打進外野 → 1.5x；全壘打 → 3-20x。

---

## 視角與場景

**主審視角**（玩家視點固定在本壘板正後方）。

```
[記分板/UFO 場外特獎]   ← 畫面最遠（top 8%）
[全壘打牆 中央 HR]       ← 12% 位置
[全壘打 左]  [全壘打 右]  ← 18% 位置
[外野安打 左] [外野安打 右]← 35% 位置
[界外 左]    [界外 右]    ← 75% 兩側
[內野滾地 + 內野飛球]    ← 內野菱形 55-70%
[打者]              [投手丘 → 打者]
▲ 主審視角（畫面最底）
```

無捕手（畫面最底就是主審視角，不畫人物）。

## 結果分區表（最終數值，平衡假設）

| 結果 | 機率 | 賠率(含本金) | 貢獻 RTP | 視覺位置 (top%, left%) |
|---|---|---|---|---|
| 沒揮（壞球/觸身） | 10% | 0x | 0 | 95%, 50% (回主審) |
| 空振 | 18% | 0x | 0 | 95%, 50% (回主審) |
| 內野滾地出局 | 14% | 0x | 0 | 70%, 50% |
| 內野飛球出局 | 10% | 0x | 0 | 55%, 50% |
| 界外 左 | 6.5% | 0x | 0 | 75%, 8% |
| 界外 右 | 6.5% | 0x | 0 | 75%, 92% |
| 外野安打 左 | 11% | 1.5x | 16.5% | 35%, 22% |
| 外野安打 右 | 11% | 1.5x | 16.5% | 35%, 78% |
| 全壘打 左 | 5% | 3x | 15% | 18%, 22% |
| 全壘打 右 | 5% | 3x | 15% | 18%, 78% |
| 正中央 HR | 2.5% | 8x | 20% | 12%, 50% |
| 場外特獎 | 0.5% | 20x | 10% | 8%, 50% (記分板) |
| **合計** | **100%** | — | **RTP 93%** |

**設計理由**：
- 失敗 65%：刺激節奏感（大部分時候空手，期待感維持）
- 小贏 22% / 1.5x：高頻小回報，玩家黏著
- 大贏 12.5%：3x / 8x / 20x 三層金字塔，讓「中外野正中」與「場外特獎」變成記憶點

## 視覺風格 lock（寫實球場風）

- **球場**：寫實夜場光，內野紅土 + 外野綠草 + 全壘打牆 + 看台
- **光源**：上方夜場燈白光，地面陰影柔和
- **配色**：草綠 #4a7c3a → #2d5023 / 紅土 #8b5a3c / 夜空 #0a1428 / 強調金 #d4a04a
- **比例**：16:9 直立式（行動 friendly，420×740 容器）
- **視角**：略低角度俯瞰打者背面 → 投手丘 → 全壘打牆 → 看台 → 夜空

跨所有生圖 sub-skill 統一：寫實球場風 + 夜場 + 主審視角。

## 角色 persona（Phase 5 對白用）

**機台廣播員**（旁白語氣）：
- 結果播報：投幣機台老派風 + 棒球轉播熱血
- 大獎時：誇張高昂（「正中央全壘打！！」）
- 失敗時：可惜但不打擊（「球被內野手接殺了！再來一球！」）

**打者 / 投手**：無台詞（純動作）。

## 平衡假設（Phase 5.1 給 game-balance-auditor）

1. RTP 93% 是長期玩家角度的回收率
2. 變異數要適中：玩家不會連 30 球都空手（pity 機制？暫不加，先看 simulation）
3. 場外特獎 0.5% 不能太低（玩家要看得到希望），也不能太高（會打破 RTP）
4. 1000 籌碼起點 + 100 平均下注 → 預期 100 球後剩 ~930（RTP 93%）

simulation 目標：跑 10000 場 100 球，看：
- 破產（餘額 ≤ 0）比例
- 賺錢（餘額 > 1000）比例
- 中位數結餘
- 最長連敗 / 連勝分布
